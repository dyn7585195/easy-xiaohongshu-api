# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-XX-XX

### Added
- 初始版本发布
- 支持小红书小程序服务端 API
- 登录认证接口 (code2Session)
- Access Token 自动管理
- 商品管理接口（获取信息、批量获取、同步、排序）
- SKU 管理接口（批量删除、上架、下架）
- 订单管理接口（创建、状态同步、获取信息、支付 Token）
- 链接管理接口（URL Link、二维码生成）
- 类目管理接口（搜索类目、获取工业类目）
- 交易管理接口（凭证核销、预约单、售后单、结算查询）
- 统一 API 调用方式
- 自动签名生成
- 完整的接口文档
